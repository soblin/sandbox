## param.m

PID制御の例題のシミュレーション. ここではプラントにはアクチュエータと電力増幅回路(1次遅れ系)からトルクが入力され, 以下のトルクから速度vへのプラントモデル

$$\begin{align}
F &= ma + cv = m\dot{v} + cv \\\\
F &= (ms + c)v \\\\
v &= \dfrac{1}{ms + c} F
\end{align}$$

を想定する. 位置決めをするのでそれを積分した位置について, PIDへのフィードバックを行う.
